Klequis' Development Guide

What is it and why is it worth paying attention to

Dear Fellow Software Developers,

I am excited to announce the launch of Klequis' Development Guide, which contains tutorials targeted toward aspiring software developers. The overall goal of the site is to help new developers acquire a well rounded skill set.

The site's current content is a series focused on creating a web development environment as a virtual machine using Linux. Its configuration draws upon my experience developing in Linux over the past 3 years.

The development environment can be created on any computer: Mac, Windows or Linux. The process is described in detail, is very pictorial and has two additional post to help you become productive if Linux is new to you.

What gets installed?

The tutorials will lead you through the process of building you development environment. It starts with installing VirtualBox and the OS for the virtual machine. This is followed by step-by-step instructions for installing the following:

Oracle VirtualBoxVirtualization software
Ubuntu 16.04Linux operating system
GNOMELinux desktop environment
ApacheWeb server
PHP 7.0.6Scripting language for web development. Both Wordpress and Drupal are built with PHP
MySQLDatabase server
AtomA hackable text editor for the 21st Century
Node.jsOpen-source, cross-platform runtime environment for developing server-side Web applications with JavaScript
GitDistributed version control system (VCS)
FileZillaFTP application
MeldVisual diff and merge tool targeted at developers
GIMPCross-platform image editor available for GNU/Linux, OS X, windows and more
ShutterFeature-rich screenshot program for Linux based OSs
MySQL WorkbenchDatabase design, creation and manipulation
ChromiumOpen-source browser for Google Chrome
GitHubWeb-based Git repository hosting service
Apache (web server), PHP (scripting language )

All the software and tools installed are free and open-source. The first 13 parts of the series are complete and cover installation of the virtual machine, Apache web server, database, PHP, phpMyAdmin and a set of key developer tools.

Future tutorials are already being written for Node.js, build tools, linting tools, and using Git & GitHub. You can get notified when these tutorial are ready by signing-up .
