Announcing: Klequis' Development Guide

Klequis' Development Guide is a series of tutorials aimed at the beginning programmer. To make the tutorials easy to follow they are very detailed, step-by-step, with lots of pictures. I get my source for ideas is the members of Tri-Valley/EastBay Free Code Camp Study Group, a group of people learning to code. While I am learning JavaScript and Node.js right alonge with them, I am hoping to teach these technologies in the future.

Current Content

Klequis' Development Guide is only a few months old and there is already a significant amount of content on it. Here is what you can find:

  • Building a Development Environment with Ubuntu GNOME 16.04: A thirteen part series that covers everything from installing the operating system as a virtual machine to installing and configuring Apache web server, MySQL database and developer tools, as well as creating your first website.
  • Using the Command Line and Understanding Permissions While the current site content is Linux focused, this tutorial applies to Mac & Linux users alike.
  • Customizing GNOME
  • Creating a VirtualBox Virtual Machine Snapshot
