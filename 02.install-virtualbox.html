This guide will show you how to install VirtualBox 5.0 on Ubuntu.

This guide was written on Ubuntu GNOME 16.04 (64-bit) and I have only tested it on that version.

VirtualBox is free software made by Oracle Corporation. A virtual machine is an 'emulation' of a particular computer system. There are different types of virtual machines. VirtualBox is a 'full virtualization VM' which is a complete operating system. As such, when you run a Ubuntu virtual machine on a Windows computer, you have the entire Ubuntu operating system running inside a window. For in-depth information on virtual machines, see Wikipedia.

Definitions

Host
The 'host' is your computer. It is the operating system which boots when you turn on your computer. In this tutorial, the host computer is Windows 8.1.
Guest
The 'guest' is the computer you run inside of VirtualBox.
Virtual Machine (VM)
The guest is also called 'the virtual machine' and I will use these terms interchangeably.

Requirements

  • A computer running Ubuntu 16.04 64-bit.
  • At least 4GB of RAM
  • At least 20GB hard drive space.
  • VirtualBox itself does not use much disk space, but the virtual machine you will create does. The initial virtual machine will take-up about 8GB of hard disk space. However, it will grow as you install software and otherwise use it. 20 GB is plenty to get started but it is a conservative number.

The Ubuntu Software Center has an install of VirtualBox. According to the reviews for that install, it is the 32-bit version. We will be using the 64-bit version by downloading it from virtualbox.org.

  1. Go to virtualbox.org and download the installation for ubuntu 16.04 ("Xenial"). You will see two links labeled i386 (the 32-bit version) and AMD64 (the 64-bit version which we will be using).
    You should go to the download page and make sure you have the most recent download. However, the links as of today are: VirtualBox for Ubuntu 16.04 Virtualbox for Windows
  2. Using Files (i.e., the file manager) go to your Downloads directory.
  3. Right-click the downloaded file (currently named virtualbox-5.0_5.0.20-106931-Ubuntu-xenial_amd64.deb) and choose Open With Software Install
  4. Enter your password when prompted
  5. Wait for the installation to complete before proceeding.
  6. Use the below command to make sure you get updates 
    $ sudo sh -c 'echo "deb http://download.virtualbox.org/virtualbox/debian $(lsb_release -cs) contrib" >> /etc/apt/sources.list.d/virtualbox.list'
  7. Download the Oracle public key so that Software Install trusts the installation 
    $ wget -q https://www.virtualbox.org/download/oracle_vbox.asc -O- | sudo apt-key add -
  8. Launch VirtualBox and check it out! You are ready to begin building your first Ubuntu virtual machine.</>
