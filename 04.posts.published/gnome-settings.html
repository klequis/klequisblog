This post is a list of all the custom GNOME settings I use and represents my preference for how I want the desktop to behave. Your preference may differ, but the below settings can make a good starting point to build upon. GNOME out of the box offers many useful settings and you can achieve much finer control by using Shell Extensions.

The below lists of settings assumes all the Shell Extensions listed in the tutorial Customizing GNOME have been installed. There are also other important settings/customizations in that post. If you have not already done so, I suggest you first follow the steps Customizing GNOME and then come back to this post.

With so many shortcuts enabled in GNOME by default it was difficult to see which were still available and to do reassignments as needed. To make things less complicated I started out by leaving only those I wanted enabled. As a result, there are some very useful actions that currently don't have a shortcut. As I make future changes to leverage these I will update this post.

Tweak Tool>Keyboard & Mouse

This is a note for the host machine. If Mouse>Show location of pointer is set to 'ON' for the host machine, the host will capture the right control key and the guest will not receive it. This means all Right-Ctrl+ combinations like copy and paste will not work on the guest.

Tweak>Keyboard and Mouse All default.

Titlebar Buttons>Minimize = ON

Extension Settings

To change an extensions settings go to Tweak Tool > Extensions. All extensions that have settings present a gear icon to the right of the description.

PutWindows
