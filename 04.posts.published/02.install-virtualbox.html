This tutorial will show you how to install VirtualBox 5.0 on Ubuntu.

VirtualBox is free software made by Oracle Corporation. A virtual machine is an 'emulation' of a particular computer system. There are different types of virtual machines. VirtualBox is a 'full virtualization VM' which is a complete operating system. As such, when you run a Ubuntu virtual machine on a Windows computer, you have the entire Ubuntu operating system running inside a window.

For in-depth information on virtual machines, see Wikipedia.

If you don't already have an established and backed-up location to write down information about your development environment such as passwords and database names, establish that now. Evernote and OneNote are both good choices.

Definitions

Host
The 'host' is your computer. It is the operating system which boots when you turn on your computer. In this tutorial, the host computer is Windows 8.1.
Guest
The 'guest' is the computer you run inside of VirtualBox.
Virtual Machine (VM)
The guest is also called 'the virtual machine' and I will use these terms interchangeably.

Requirements

  • A computer running Ubuntu 16.04 64-bit.
  • At least 4GB of RAM
  • At least 20GB hard drive space.
  • VirtualBox itself does not use much disk space, but the virtual machine you will create does. The initial virtual machine will take-up about 8GB of hard disk space. However, it will grow as you install software and otherwise use it. 20 GB is plenty to get started but it is a conservative number.

The Ubuntu Software Center has an install of VirtualBox. According to the reviews for that install, it is the 32-bit version. We will be using the 64-bit version by downloading it from virtualbox.org.

Go to virtualbox.org and download the installation for ubuntu 16.04 ("Xenial"). You will see two links labeled i386 (the 32-bit version) and AMD64 (the 64-bit version which we will be using).

Install VirtualBox

Using Windows? See How to Install VirtualBox on Windows.

  1. Using Files (i.e., the file manager) go to your Downloads directory.
  2. Right-click the downloaded file (currently named virtualbox-5.0_5.0.20-106931-Ubuntu-xenial_amd64.deb) and choose .
  3. In Software, click .
  4. Enter your password when prompted.
  5. Wait for the installation to complete before proceeding.
  6. Use the below command to make sure you get updates. (I suggest you copy/paste it.) 
    $ sudo sh -c 'echo "deb http://download.virtualbox.org/virtualbox/debian $(lsb_release -cs) contrib" >> /etc/apt/sources.list.d/virtualbox.list'
  7. Download the Oracle public key so that Software Install trusts the installation
    $ wget -q https://www.virtualbox.org/download/oracle_vbox.asc -O- | sudo apt-key add -
  8. Launch VirtualBox and check it out! You are ready to begin building your first Ubuntu GNOME virtual machine.

I ran into a problem getting Ubuntu updates and found a cure as follows.

Check if your machine is experiencing the issue.

$ sudo apt-get update

If you see an error similar to the below you will not be able to get VirtualBox updates.

W: GPG error: http://download.virtualbox.org/virtualbox/debian xenial InRelease: The following signatures couldn't be verified because the public key is not available: NO_PUBKEY A2F683C52980AECF
W: The repository 'http://download.virtualbox.org/virtualbox/debian xenial InRelease' is not signed.
N: Data from such a repository can't be authenticated and is therefore potentially dangerous to use.

To fix this, copy the key, in my case A2F683C52980AECF, and execute the below command.

$ sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys A2F683C52980AECF

If it is successful, you will see a message something like this.

Make a Snapshot

Don't skip this step!

As you are install software on your fresh new VM things can go wrong and not all of them are easy to correct. Creating a snapshot at the end of each major change to your VM will give you a way to go back to a state when things were working so you can try again. Go to this page How to Create a Snapshot With Oracle VirtualBox and make a snapshot now!

Congratulations! VirtualBox is installed and you are ready but build your first virtual machine.
