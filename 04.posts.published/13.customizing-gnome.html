This tutorial will show you how to make some common customizations to the GNOME desktop environment. Out of the box, GNOME may not work exactly the way you want to. However, GNOME is very customizable. The tutorial will introduce you to customizing GNOME using Settings, Tweak Tool, GNOME Shell Extensions and dConf Editor.

Using Multiple Monitors

This first task is not a GNOME customization, but this seems like a good place to put it anyway. If you have more than one monitor and are doing development on your VM, you will want the VM to use both monitors as working across two machines (host and VM) is cumbersome and lot's of extra clicks. Follow these steps to setup multiple monitors. I have never tried to use more than two monitors, but the process should be the same.

In the VirtualBox Manager, make sure the VM is 'Powered Off'.

Enable Dark Theme & Change Background

If you are like me you don't like a bright white screen in your eyes for hours on end. GNOME allows you to quickly darken things up.

Global Dark Theme

Open "Tweak Tool".

Select the Appearance tab.

Set 'Global Dark Theme' to 'ON'.

Changes will take effect as you close and reopen applications.

Change Background

Right-click on the desktop and choose .

Click on the picture captioned "Background".

You can choose between Wallpapers, Pictures and Colors. If you are as into dark as I am, try the grayscale leaves with dark background.

Do you want more darkness? If so, install the Chromium extension Dark Reader.

Using GNOME Shell Extensions

Shell Extension are a powerful way to modify the GNOME desktop. Follow along below, and then go explore the list of extensions to find those that are most helpful to you.

Setup FireFox

When you first point FireFox at the Extensions page there will be a red error message at the top. This is fixed by installing a FireFox add-on.

Go to https://extensions.gnome.org/ from which you can search the list of extensions.

On the bar at the top of the page click .

If for any reason the above doesn't work, go to the hamburger menu at the top right of FireFox and select Add-ons. Search for 'GNOME Shell Extensions' and enable it.

A dialog will pop-up. Click .

Refresh the page and the red message should go away.

Installing a Shell Extension

The steps below show how to enable the Dash to Dock extension. Following that is a table that lists other extensions that I find useful. Use the same steps to install them.

Select Extensions at the top of the page.

Search for 'Dash to Dock'.

Click on Dash to Dock.

Click 'OFF' next to Dash to Dock tile. This will switch the state to 'ON'.

A modal pop-up will appear. Click .

Open 'Tweak Tool' and click on the 'Extensions' tab.

Find 'Dash to dock' in the list and click on the gear button.

Explore the available settings to make Dash to Dock work the way you prefer.

Not all of the GNOME Shell Extensions have settings, but many of them do, and the settings can be quite extensive. Be sure to explore!

List of Other Useful Extensions

These are the other extensions I have installed on my machine. There are many more that you may find useful. Feel free to add them to the post comments.

Extension Name Description
AlternateTab By default, GNOME will group windows for each application in the Alt-Tab UI with a drop-down arrow to get to individual windows. I find this creates stressful finger gymnastics. AlternateTab un-groups the individual windows.
Harddisk Shows hard disk activity in the top bar.
Launch new instance By default, if an application is already running and you go to the Dash and click on the application, GNOME will bring up the running instance. This extension changes that so a new instance is launched.
PutWindows Provides many options for managing windows including keyboard shortcuts, window sizes, selecting windows and more.
Recent(Item)s Puts an icon in the top bar with a dropdown containing a list of recent files. I have it configured so that Super-f activates it.
Service Panel Puts and icon with dropdown in the top bar which shows the state of various services such as a web server and MySQL.
User Themes Allows you to put a shell theme in ~/.themes. Putting themes here makes it available for the current user instead of all users of the machine.
No Toplef Hot Corner I really hate it when I move my mouse to the top left of the screen to click on something and the Overview pops-up. This extension turns off the 'hot corner'. Since the Super key does the same thing, the hot corner isn't needed.

Opinion

Different people have different preferences. I certainly have mine. However, I think people largely prefer what they are used to and it can be tempting to install Extensions that make GNOME more Windows like or more Mac like. I suggest you don't do that. GNOME is different and it is really quite good. Spend some time learning how to navigate GNOME, especially how to do so using the keyboard as this is one of its strengths. Most keyboard shortcuts are customizable and we will be doing some of that in this post. Give GNOME's approach a couple of months. If you still don't like it then customize to suite you.

Printers

I'll just quickly mention that GNOME provides a nice UI to printer options. I was dismayed when I learned that my inkjet printer was using color ink to print grayscale. I set two defaults here:

  1. Color > Output Mode = Black Only Grayscale
  2. Image Quality > Print Quality = Draft.

Most of my printing is for my own use so draft quality is good enough.

Keyboard

Some of the extensions you install will have conflicting keyboard short-cuts and you can end-up with two short-cuts that do the same thing. As I'm writing this post I resolved conflicts by disabling everything I don't use and then making a list of all the active short-cuts and actions. Some of these changes will not be evident in the screenshots.

If you are a serious keyboard user, and there are advantages to being one, you will want to make a list of your keyboard shortcuts. This will allow you to get an organized and memorable set of shortcuts, as well as be a source if you should need to build a new machine.

Start in Settings > Keyboard on the 'Shortcuts' tab. If you are using PutWindows it is especially important to disable everything on the Windows tab.

It is easy to over-look the message at the bottom of the screen that says '...press Backspace to clear'. Doing so will disable the shortcut.

If your host machine is Ubuntu, beware of the setting 'Show location of pointer' as when set to 'ON' the guest will not receive the Ctrl key. As a result no Ctrl+* shortcuts will work including those for copy and paste.

Firewall Configuration

As mentioned previously, VirtualBox has already secured your VM. However, I include the firewall configuration here for those who are using Ubuntu as the host.

There is more to configuring a firewall than I can cover in this post. However, I'll discuss the main configuration options.

Profile: If you have a laptop you can have different settings for Home, Public and Office. For many people, one profile will be enough.

Status: 'ON/Off' - turn it ON.

Incoming: Set this to 'Deny'. If you were configuring a web server that needed to be accessible from the internet or intranet you would set this to 'Allow', but don't! We want our development machine as isolated as possible, so set this to 'Deny'. Doing so will result in all incoming requests being denied.

Outgoing: Set this to 'Allow'. This allows request from your browser to the outside world. You need this to access the internet.

Summary of Settings

  • Status = ON
  • Incoming = Deny
  • Outgoing = Allow

Tweak Tool

We have already see the Tweak Tool when setting the Global Dark Theme and working with extension settings. There is a lot more you can do with Tweak. It is well worth a few minutes of exploration.

Minimize and Maximize Buttons!!

You likely have already noticed and been frustrated by the lack of minimize and maximize buttons in GNOME. The good news is you can turn them on.

Increase Mouse / Pointer Size

There are many, many mouse/cursor themes available. One good site to find them on is GNOME-LOOK.ORG. Look under 'Mouse Themes' for one that appeals to you. Some of them say 'Multisized' and some do not. My guess is that you need to pick a multi-sized one, but I have not checked this out. For this example, I'm going to use a fun one named Ultimate-Black. You need to have a Dropbox account to download it.

Dropbox is very useful tool for moving files between guest and host machines. While you can setup folders that are shared between the guest and host, I find Dropbox most convenient.

Download the package and unzip it.

Copy the cursor theme's folder to /usr/share/icons.

$ cd ~/Downloads
$ sudo mv Ultimate-Black /usr/share/icons

Open the Tweak Tool.

In the Appearance tab, set Cursor = Ultimate-Black (or the name of you theme if you are using a different one).

If Tweak is already running you need to close and reopen. Tweek only reads changes when it is opening.

Open dConf Editor and search (ctrl+f) for 'cursor-size'.

Double-click on the value to change it. You can set the size to 24, 32, 40, 48, 56 or 64.

Set Files Preferences

Finally, check-out the many options for Files. I made the below changes to suit my preference.

  • Views: View new folders using = List view
  • Views: Sort folders before files = Checked
  • Views: Show hidden and backup files = Checked
  • Display: Set all 3 drop-downs to 'None'
  • Preview: Show thumbnails = Always
