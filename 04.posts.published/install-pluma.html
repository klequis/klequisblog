Prior to Ubuntu 16.04 I was a big fan of 'gedit' which is a text editor that comes with Ubuntu. With Ubuntu 16.04 the menu UI in gedit changed and for a while I didn't want to use it, so I used Pluma instead. Then I switched to GNOME and lots of apps had menu UI similar Pluma in Ubuntu 16.04, which I slowly got used to and am back to using gedit full-time. Pluma is almost identical to gedit pre-Ubuntu 16.04 and you can use it if you prefer that UI.

Let's install Pluma...

This tutorial is writing using Ubuntu with the Unity desktop environment. Installation under the GNOME desktop environment is very similar and you should be able to use the below instructions with GNOME.

Installation

Click the Ubuntu Home button

Type 'software' and click the 'Ubuntu Software' icon

In the search box type 'pluma'

Click

When prompted to authenticate, enter you account password. This is the same password you use to log into Ubuntu.

When installation is complete you should see an icon for Pluma in the Launch Bar. If not, click the home button and search for it.

That it! Pluma is pretty straight-forward to install.
