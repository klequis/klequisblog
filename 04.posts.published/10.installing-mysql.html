This guide will show you how to install MySQL Server and Client on Ubuntu GNOME 16.04. MySQL is an open-source relational database management system. It is one of the most widely used relational database management systems among free and non-free systems. MySQL is free for open-source projects and for non-commercial use.

MySQL Server
MySQL Server is the database engine and storage.
MySQL Client
MySQL Client is a command line application that communicates with MySQL Server. Using MySQL client you can create database objects, issue queries and much more.

If you want to use a GUI based client for managing the database, we will be installing phpMyAdmin later in the tutorial Installing phpMyAdmin.

We will be installing the latest version of MySQL which as of this writing in June 2016 is 5.7.13.

These instructions are for a machine that has never had any version of MySQL installed on it. If your machine already has MySQL installed, do not follow these instructions. Consult the MySQL installation documentation.

Preperation

This tutorial is part of the series Building a Ubuntu GNOME 16.04 Development Machine. If you have been following the series your machine is ready for MySQL installation. Make sure you have the password you created in Introduction to LAMP.

The root user us the 'superuser' for MySQL. The root user can make any and all changes to any of the databases you instance of MySQL will host. Since your development machine, as built following this series, will not be accessible from the internet, the complexity of the password is not important. It is there for two reasons: 1) to make sure you think about it when working as root and 2) to give you the full experience of working with a database server which includes setting and entering passwords.

It is possible to configure MySQL with 'blank' as the password but please don't.

Install MySQL

To install MySQL Server & Client, execute the below command. Since the command does not specify a version number, the most recent version will be installed.

sudo apt install mysql-server

Enter 'y' when prompted.

Enter the password you created for the root user. Use tab to navigate to 'OK' and press enter.

Re-enter your password.

When installation is complete test MySql with the below command. This command may take a while to execute.

The output will look like the picture but some numbers may be different.

sudo netstat -tap | grep mysql

If the above command fails, try restarting MySQL with the below command and then try it again.

sudo systemctl restart mysql.service

Other Useful MySQL Commands

There are many MySQL commands which you can learn from the MySQL documentation. Here are a few to get you started.

Check the MySQL version.

$ mysql --version

Extended MySQL version information.

$ mysqladmin -p -u root version

Check MySQL Server status.

(Press 'q' to exit status.)

$ service mysql status

Stop MySql.

$ sudo systemctl stop mysql.service

Start MySql.

$ sudo systemctl start mysql.service

Reseting Root Password

I tried three different methods for resetting the root password and none of them worked. It is best to put your password somewere safe!

Congratulations! MySQL is installed and ready for use.
