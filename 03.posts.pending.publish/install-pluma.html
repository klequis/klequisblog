Prior to Ubuntu 16.04 I was a big fan of 'gedit' which is a text editor that comes with Ubuntu. However, with Ubuntu 16.04 the UI in gedit has changed in ways that make it difficult to use. Therefore, we are going to use 'Pluma' which is almost identical to gedit pre-Ubuntu 16.04. Let's install Pluma...
    1. Click the Ubuntu Home button
    2. Type 'software' and click the 'Ubuntu Software' icon
    3. In the search box type 'pluma'
    4. Click
    1. When prompted to authenticate, enter you account password. This is the same password you use to log into Ubuntu.
    1. When installation is complete you should see an icon for Pluma in the Launch Bar. If not, click the home button and search for it.
