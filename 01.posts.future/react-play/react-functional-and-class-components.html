This example project introduces React 'class' components and illustrates the differences between class and functional components. This will be a brief introduction. A later example will illustrate more advanced features that are available when using classes such as state and lifecycle events.

What is a Functional Component

In the example projects so far we have been using functional components. A functional component is a react component that exports a function. It does not extend a React class as a class component does.

What is a Class Component?

In React, a class component is a component that extends the class React.Component. Class components can do things that functional components cannot, such as maintain state and use React lifecycle events.

What Makes a Functional Component a Functional Component?

Looking at the file Auto01.jsx there are several features that make it a react component:

  • React is imported
  • It contains a function that returns JSX
  • The function is exported using export default
import React from 'react';
const Auto01 = function Auto(props) {
  return (
    <div>
      <h3>Auto01 - is a functional component</h3>
      <p>My car is a {props.make}</p>
    </div>
  )
}
export default Auto01;
