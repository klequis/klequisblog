There is seemingly an unlimited number of packages you can add to Atom to customize the way it works. In this post, I'm going to show you the packages that I find most useful. We will be doing installation via the command line and it will only take a few minutes.

If you do not have Atom installed, go to http://atom.io and install it now.

We will install the following packages

  1. color-picker
  2. linter
  3. linter-csslint
  4. linter-eslint
  5. linter-htmlhint
  6. open-in-browser
  7. split-diff
  8. webbox-color

To install the above packages open terminal and use the following command

apm install linter linter-csslint linter-eslint linter-htmlhint open-in-browser split-diff webbox-color seti-ui color-picker

If Atom is open, close and reopen it.

Here are the editor settings I prefer

In Atom choose Edit>Preferences and then the Editor tab on the left.

Change settings to match the below.

  1. Scroll Past End = checked
  2. Show Indent Guide = checked
  3. Show Invisibles = checked
  4. Soft Tabs = checked
  5. Soft Wrap = checked
  6. Tab Length = 2

A nice theme with icons which show file type

The Seti theme was installed when we use the 'apm' command above. It needs to be activated.

  1. In the settings page, choose 'Themes' on the left
  2. Set UI Theme = Seti
  3. Set Syntax Theme = Base16 Tomorrow Dark
