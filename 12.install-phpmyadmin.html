This guide will show you how to install phpMyAdmin. phpMyAdmin is free software for managing MySQL via a web interface.

Installation

To install phpMyAdmin you need to install three packages: phpmyadmin itself, php-mbstring and php-gettext. Execute the following command to install all three.

sudo apt-get install phpmyadmin php-mbstring php-gettext

Enter 'y' when prompted and press Enter.

In the installation UI a red rectangle indicates which part of the UI has focus. Use the Arrow keys to move between items in the list box containing 'apache2', and 'lighttpd' and use Tab to move from the list to the button.

Make sure the red rectangle is on 'apache2' and press Space to select it.

Press Tab to move focus to the and then press Enter.

The next question will ask if you want to "Configure database for phpmyadmin with dbconfig-common. Select and press Enter

In this step you are asked to provide a password that phpMyAdmin will use to access MySQL. The phpMyAdmin installation will create a new MySQL user named 'phpmyadmin' which will use the password you provide to log into MySQL.

As I have mentioned earlier in this series, this is a Virtual Machine that cannot be accessed from the internet or even its host, so you can be use a simple password. I use the same 4 character password that I use for the MySQL root account.

Confirm the password

Launch & Log Into MySQL

There are two ways to launch phpMyAdmin. You can open a browser and navigate to http://localhost/phpmyadmin. phpMyAdmin can also be launched like an GNOME application by pressing Super and typing 'phpmyadmin'. phpMyAdmin will appear like any other installed application but when launched simply opens a browser and navigates to http://localhost/phpmyadmin.

Launch phpMyAdmin using either method.

Log in with 'root' as the username and the password you created when installing MySQL
