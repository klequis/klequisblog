This guide will show you how to install phpMyAdmin. phpMyAdmin is a free software for managing MySQL via a web interface.

Installation

To start installation, execute the below comand.

sudo apt-get install phpmyadmin php-mbstring php-gettext

Enter 'y' when prompted and press Enter.

pic.001.install.phpmyadmin.png

You will be asked to select either apache2 or lighttpd. Select apache2 by making sure the red rectangle is in the box for apache 2 and then press Space to mark it as selected.

Use your arrow keys to move the red rectangel between apache2 and lighttpd if needed. Use Tab to move to the and then press Enter to continue.

pic.002.select.web.server.png

The next question will ask if you want to "Configure database for phpmyadmin with dbconfig-common. Select and press Enter

pic.003.dbconfig-common.yes.png

In this step you will provide a password for phpMyAdmin to use to access MySQL. The phpMyAdmin installation will create a MySQL user named 'phpmyadmin' which will use the password you provide to log in.

Again, as I have mentioned earlier in this series, this is a Virtual Machine that cannot be accessed from the internet or ever its host, so you can be real simple with the password. I use the same 4 character password that I use for MySQL.

pic.004.mysql.password.png

Confirm the password

pic.004.mysql.password.png

Launch & Log Into MySQL

Open a browser and navigate to http://localhost/phpmyadmin

Log in with 'root' as the username and the password you created when isntalling MySQL
