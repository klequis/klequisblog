This guide will show you how to install MySQL. MySQL is a database server. In it you can create and query databases. The latest version of MySQL is version 5.7.13.

These instructions are for a machine that has never had any version of MySQL installed on it. If your machine already has MySQL installed, do not follow these instructions. Consult the MySQL installation documentation.

Preperation

This is the xxx part of the series SERIESNAME. If you have been following the series your machine is ready for installation and you have only one thing to prepare - the password you will use for the MySQL root user.

The root user us the 'superuser' for MySQL. The root user can make any and all changes to any of the databases you instance of MySQL will host. Since your development machine, as built following this series, will not be accessible from the internet, the complexity of the password is not important. It is there for two reasons: 1) to make sure you think about it when working as root and 2) to give you the full experience of working with a database server which includes setting and using passwords.

It is possible to configure MySQL with 'blank' as the password but please don't
