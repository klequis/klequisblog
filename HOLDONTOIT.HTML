Setup GitHub

Go to github.com and create an account.

Once you verify your email address you will be redirected to a new page. Click the button.

pic.003.github.start.project.png

I am going to create a repository named 'git-github-tutorial'. You can use a different name. However, because I will be referring to 'git-github-tutorial' in the steps below it will be easier to follow the steps if you use git-github-tutorial as well.

Enter 'git-github-tutorial' as the repository name.

Keep the default of Public. Creating a private repository requires a paid account.

Check 'Initialize this repository with a README'. All projects should have a readme.md file regardless of how simple the project or readme is.
