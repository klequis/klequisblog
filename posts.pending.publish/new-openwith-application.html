This guide will show you how to create a shortcut for an application that does not appear in the context (i.e., right-click) menu under Open With or Open With > Other Application... Although the process below is not very hard, this is one area where I wish there was a simplier and quicker method. Fortunately, the majority of applications installed create their own. Unfortunately Atom text editor did not, so I use that in this example.

Title

Create a new file named 'atom.desktop' in /usr/share/applications.

$ sudo gedit /usr/share/applications/atom.desktop

Paste the below text into the new file.

[Desktop Entry]
Name=Atom
Comment=A hackable text editor for the 21st Century.
GenericName=Text Editor
Exec=/usr/share/atom/atom %U
Icon=atom
Type=Application
StartupNotify=true
Categories=GNOME;GTK;Utility;TextEditor;Development;
MimeType=text/plain;

Save and close the file.

Right click on a file, a .txt or .html would be a good choice, and try to open it with Atom
