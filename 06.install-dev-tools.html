Whether you are a developer, a designer or both, you need a set of tools to get the job done. In this guide we will install tools that are needed by everyone who does website development.

All the below applications are FREE!

We will install the following tools

FileZilla FileZilla is an FTP application. As a web developer you will most often use it to move files between your local machine and your hosting account.
Meld Meld is a visual diff and merge tool targeted at developers. Meld helps you compare files, directories, and version controlled projects. It provides two and three-way comparison of both files and directories, and has support for many popular version control systems. Meld helps you review code changes and understand patches. It might even help you to figure out what is going on in that merge you keep avoiding.
GIMP Gimp is an image manipulation program similar to Photoshop.
Atom Atom is a text editor that is well suited to web development.
Shutter Shutter is a feature-rich screenshot program.
Chromium Chromium is a version of Google's Chrome browser that runs on Linux.

Is Chromium a developer tool? I would say 'yes'. If you are not familiar with Chromium's developer tools, install it, then go to a website using Chromium and press F12 to get a taste of what is there.

Using Favorites

Ubuntu comes with a selection of applications pinned in Favorites. As a developer I don't use many of them, I use the developer applications we just installed, so I remove many of the default favorites and pin the ones I use most.

I'll also add that I generally don't use favorites to launch applications because it is easier to press Super and type a few letters. I do use it as a status view of running applications. For example, press Super, type 'term' and press Enter to launch Terminal.
