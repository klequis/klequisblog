This guide will show you how to install Deluge BitTorrent Client.

Definitions

BitTorrent
A peer-to-peer file transfer protocol for sharing large amounts of date over the Internet. When you download via BitTorrent the data comes from multiple sources and is reconstructed into one download.
BitTorrent Client
Software that runs on you computer which is used for downloading via BitTorrent.
Pear to pear
A computer network in which each computer can act as both client and server. In the context of BitTorrent, this means that each computer can be the source a single download to many other computers and can fetch a single download from many other computers. In peer-to-peer, there is not central server.

Is using BitTorrent safe

Historically, there have been some cases where BitTorrent clients have installed malware. My expertise is not computer security. However, I have been using BitTorrent for many years and have not had a problem. As always, you should research software that you are installing from the Internet.
