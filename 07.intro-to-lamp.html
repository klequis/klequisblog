LAMP stands for Linux, Apache, MySQL & PHP. It is a software stack comprised completely of free and open-source software. Wikipedia has an excellent article that is not too long, so take a minor detour, give it a read, and then come back.

LAMP provides you with a web server to host your sites, a database to store your data and the PHP scripting language. It is an excellent environment for WordPress, Durpal and any other PHP system.

As noted in the introduction to this series the development environment we are building will work for the technologies just mentioned as well as Node.js and JavaScript. These technologies are installed later in this series.
