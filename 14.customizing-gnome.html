You could waste a lot of time on this ... Change Super from left to right 1. open Tweak Tool 2. Select the Keyboard and Mouse table 3. Change 'Switch between overview and desktop' to 'Right super' Set Files Preferences 1. Click Files on the top bar and then Preferences 2. Set 'View new folders using:' to List view 3. Set Sort folders before files = True 4. Set Show hidden and backup files = True 5. Select the Display tab 6. Set all 3 drop-downs to 'None' 7. Select the Preview tab 8. Set 'Show thumbnails' to 'Always'
