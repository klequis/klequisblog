GNOME is very customizable and you could waste a lot of time making it look the way you want. I guess 'waste' isn't really right for everyone, a designer or usability person might be very interested in how the UI can be changed. Whichever way you want to look at it, you can sink more or less time into it.

Below are instructions for ways I customize my machine to make it more livable for me. The changes are not interdependent so you can choose to do any number of these.

Using 2 or More Monitors

This first task is not a GNOME customization but this seems like a good place to put it anyway. If you have more than one Monitor and are doing development on your VM you will want the VM to use both monitors as working across two machines (host and VM) is combersome and lot's of extra clicks. Follow these steps to setup 2 monitors. I have never tried to use more than 2 but it looks like the steps are the same.

In the VirtualBox Manager, make sure the VM is 'Powered Off'.

Go to > and set the 'Monitor Count' to 2.

Press .

Power on the VM.

Next you need to enable the second monitor. Go to the VirtualBox menu on the VM's window, and select >>.

It is possible at this point that as you move you mouse between monitors it gets stuck. This is because the left monitor is right and the right is left, or at least Ubuntu thinks so. Here is how to fix it:

Right-click on the desktop and click .

Click .

Drag the monitors to switch their position.

Enable Dark Theme & Change Background

If you are like me you don't like a bright white screen in your eyes for hours on end. GNOME allows you to quickly darken things up.

Global Dark Theme

Open "Tweak Tool".

Select the Appearance tab.

Set 'Global Dark Theme' to 'ON'.

Changes will take effect as you close and reopen applications.

List of Other Useful Extensions

These are the other extensions I have installed on my machine. There are many more that you may find useful. Feel free to add them to the post comments.

Extension Name Description
AlternateTab By default, GNOME will group windows for each application in the Alt-Tab UI with a drop-down arrow to get to individual windows. I find this creates stressfull finger gymnastics. AlternateTab ungroups the individual windows.
Harddisk Shows harddisk activity in the top bar.
Launch new instance By default, if an application is already running and you go to the Dash and click on the application, GNOME will bring the running instance will be brought to the top of the z-order. This extension changes that so a new instance is launched.
PutWindows Provides many options for managing windows including keyboard shortcuts, windows sizes, selecting windows and more.
Recent(Item)s Puts an icon in the top bar with a dropdown containing a list of recent files. I have it configured so that Super-f activates it.
Service Panel Puts and icon with dropdown in the top bar which shows the state of various services such as a web server and MySQL.
User Themes Allows you to put a shell theme in ~/.themes. Putting themes here makes it available for the current user instead of all users of the machine.

Opinion

Different people have different perferences. I certaintly have mine. However, I think people largely prefer what they are used to and it can be tempting to install Extensions that make GNOME more Windows like or more Mac like. I suggest you don't do that. GNOME is different and it is really quite good. Spend some time learning how to navigate GNOME, especially how to do so using the keyboard as this is one of its strengths. Most keyboard shortcuts are customizable and we will be doing some of that in this post. Give GNOME's approch a couple of months. If you still don't like it then customize to suite you.
